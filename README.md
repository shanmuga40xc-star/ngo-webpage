# InAmigos Foundation Website

A modern and responsive NGO awareness website designed for InAmigos Foundation as part of an internship project.

## Features

- Responsive and user-friendly design
- Hero section highlighting the NGO's mission
- Donation and volunteer engagement sections
- Impact statistics and beneficiary showcase
- Community development and educational initiatives
- International Day of Happiness awareness section
- Modern green-themed UI inspired by NGO and social impact platforms
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome Icons

## Project Objective

The objective of this project is to create an engaging digital platform that promotes the mission, initiatives, and social impact of InAmigos Foundation while encouraging community participation through volunteering and donations.
